# Console-based-Library-Management-System
This is a  console-based library management system implemented in Java. The system allows users to perform  operations such as adding books, searching for books, borrowing books, and returning books. The project is intended as a starting point for understanding and building more complex library management systems.
Features :

Add books to the library with unique IDs, titles, and author names.
Search for books by their IDs.
Borrow books by associating them with borrower information.
Return borrowed books and update the borrower records.
User-friendly console interface for interactions.
Getting Started
Clone or download the repository.
Compile the Java files using your preferred Java compiler.
Run the LibraryManagementSystem class to start the console interface.


Usage
Follow the on-screen prompts to interact with the library management system:

Choose options to add, search, borrow, or return books.
Input relevant information such as book details and borrower information.
Perform actions and observe system responses.
